# Global Screen Rules

Every screen starts with a header.

Every screen has a purpose statement.

Every screen has at least one KPI or insight.

Every screen has one primary action.

Every screen supports Arabic and English.

Every screen supports mobile and desktop.

---

# Mobile Wireframe Rule

Mobile screens use:

Top header

Compact hero

Vertical cards

Bottom navigation

Sticky language switch

---

# Desktop Wireframe Rule

Desktop screens use:

Sidebar

Topbar

Main canvas

Optional right insights panel

---

# Golden Wireframe Rule

No screen should be built before its structure is defined here.
