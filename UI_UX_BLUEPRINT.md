# Purpose

This document defines the complete user experience and interface blueprint of the AI Transformation Center platform.

No screen should be developed before its structure is approved here.

---

# Product Experience Goal

The platform must feel like an executive command center, not a normal website.

The first 30 seconds must communicate:

Clarity

Power

Trust

Innovation

Execution Readiness

# Interaction Rules

Every page must have:

Page title

Short description

Primary KPI

Main content

Secondary insights

Clear action

---

# Visual Rules

Large cards

Rounded corners

Soft shadows

Clear spacing

Premium colors

Minimal text

Strong hierarchy

---

# Navigation Rules

The user must reach any main page within one tap on mobile and one click on desktop.

Dashboard is always first.

Settings is always last.

Language switch is always visible.

---

# Presentation Mode

Future mode for presenting to leadership.

Hides editing tools.

Shows only executive visuals.

Large typography.

High contrast.

---

# Golden UX Rules

The platform must be impressive in 30 seconds.

The dashboard must explain the whole strategy.

The user must never feel lost.

Every number must have meaning.

Every section must support Arabic and English.

Every screen must feel executive ready.
